# SAST Semgrep Demo

Projet de démonstration de l’intégration d’un outil SAST (Semgrep)
dans un pipeline CI/CD GitHub Actions.

## Vulnérabilités présentes
- SQL Injection
- Command Injection
- XSS

## Objectif
Illustrer la boucle de rétroaction :
Code → Analyse SAST → Feedback → Correction → Re-analyse
